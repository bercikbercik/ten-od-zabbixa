## Import do Grafany

1. Otwórz Grafanę → **Dashboards → Import**.
2. Wybierz plik `Przykładowy dashboard 3-1767029667448.json`.
3. Ustaw datasource na  `Zabbix`.
4. Dashboard powinien być gotowy do użycia.
